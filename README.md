# シナリオ画面メーカー（FGO風・非公式ファンメイドツール）

Fate/Grand Order 風のシナリオ画面（背景・立ち絵・セリフウインドウ・選択肢など）を組んで、1600×900のPNG画像として書き出せるブラウザツールです。
インストール不要・アカウント登録不要・外部送信なし。画像やテキストはすべてブラウザ内で処理されます。

> 本ツールは個人が趣味で制作した非公式のファンメイドツールです。TYPE-MOON / Aniplex / Delightworks等、Fate/Grand Orderの権利者とは一切関係ありません。

🔗 **Web版**: https://scene-maker-hikoushiki.github.io/fgo-scene-maker/

## できること

- **背景**: 複数登録して切り替え、ズーム＋ドラッグでのパン位置調整、背景ごとの色調（なし/グレースケール/セピア）
- **キャラクター（最大15人）**
  - ドラッグで移動、選択時に四隅のハンドルをドラッグして拡大縮小（掴んだ角の対角側を支点に、中心・端に吸着）
  - 左右反転、シルエット化、ホログラム化、グレースケール化、透明度スライダー
  - 端のフェード（上下左右を個別にON/OFFできるワイプ式フェード）
  - 差分画像（同キャラの別ポーズ・別衣装を追加し、共通の位置/サイズ調整＋差分ごとの微調整オフセットを両立）
  - 表情差分シート（本体＋顔差分グリッドを1枚にまとめた画像から、範囲・グリッドを指定して表情を切り替え。顔位置の自動検出／ドラッグ配置に対応）
  - サーヴァント退去エフェクト（加算合成の動画に合わせて立ち絵を進行度に応じてフェードアウト。進行度・フェード開始/終了・位置/サイズ補正・色相補正をスライダーで調整可能）
  - 重なり順（前面/背面）の変更
- **演出**: 最前面以外のキャラクターを暗くする、UI以外（背景＋キャラクター）をグレースケール／セピア化
- **セリフウインドウ**: 話者名・本文、名前欄の表示切替、最前面キャラの名前との自動連動、ルビ（ふりがな）対応、文字サイズ・文字色（グラデーション表示）
- **選択肢**: 表示切替、1〜3個の切り替え、選択肢ごとの個別カラー（グラデーション表示）
- **UI要素**: セリフウインドウ全体、SKIP/LOG/AUTO/AUTO選択中/NEXTを個別にON/OFF
- **書き出し**: ファイル名を指定してPNG画像として保存
- **プロジェクト保存/読み込み**: 画像を含む全設定を単一のJSONファイル（`.fgoscene.json`）として保存・復元
- **モバイル対応**: 画面幅860px以下では縦積みレイアウトになり、プレビューが画面上部に追従表示（スイッチでオン/オフ可能）。タッチ操作向けにボタン類の当たり判定を拡大

## 使い方

Web版を開くだけで使えます。ローカルで動かしたい場合は下記を参照してください。

1. このリポジトリをダウンロード（Code → Download ZIP、または`git clone`）して展開する
2. `index.html` をブラウザ（Chrome / Edge推奨）で**ダブルクリックして開く**
3. 左側のプレビュー画面に、右側のコンソールパネルから素材を追加していく
   - **背景**：「背景を追加」から画像をアップロード
   - **キャラクター**：「立ち絵を追加」から画像をアップロード（最大15人）。プレビュー上でドラッグ移動、選択して四隅のハンドルで拡大縮小できる
   - **セリフウインドウ**：話者名・本文を入力。ルビは `単語《よみ》` や `｜任意の範囲《よみ》` で振れる
   - **演出／選択肢／UI要素**：各パネルのスイッチやスライダーで見た目を調整する
4. コンソール下部のタブで「画像保存」「プロジェクト」を切り替えられる
   - **画像保存**：ファイル名を指定して1600×900のPNGとしてダウンロード
   - **プロジェクト**：作業内容をまるごと`.fgoscene.json`として保存・あとから読み込み可能（読み込み時は現在の内容が失われる旨の確認ダイアログが出ます）
5. スマホなど狭い画面では、プレビュー右上の「プレビュー」スイッチでプレビュー表示を折りたたみ、コンソール操作をしやすくできる

## ファイル構成

```
fgo-scene-maker/
├─ index.html          … 画面本体
├─ style.css           … 見た目
├─ script.js           … 動作ロジック（Canvas描画・操作・PNG書き出し・プロジェクト保存/読込）
├─ assets-data.js       … UI画像＋退去エフェクト動画（自動生成・埋め込み済み）※動作に必須
├─ font-data.js         … フォント（自動生成・埋め込み済み）※動作に必須
├─ build_assets.py      … assets/・fonts/ の中身から上記2つを再生成するスクリプト
├─ assets/              … UI画像・動画の元ファイル
│   ├─ textbox.png          … テキストウインドウ（名前欄は別レイヤー）
│   ├─ name_box.png         … 名前欄タブ
│   ├─ lines_box.png        … 選択肢のテキストボックス
│   ├─ departure.mp4        … サーヴァント退去エフェクト
│   ├─ skip_button.png      … SKIPボタン
│   ├─ log_button.png / auto_button.png / next.png … 右下のミニアイコン
│   └─ favicon.png          … ブラウザタブ用アイコン
└─ fonts/
    └─ GenEiLateMinN_v2.woff2   … セリフ用フォント
```

`assets/`・`fonts/`内のファイルは全てオリジナル素材です。`assets-data.js`・`font-data.js`はこれらを`build_assets.py`で埋め込み生成したもので、`index.html`をローカルで直接ダブルクリックして開いた場合でも（ローカルサーバー無しでも）問題なく動作するようにしています。

## 素材を差し替えたいとき

`assets/` や `fonts/` の中身を新しいファイルに差し替えたら、ターミナルで以下を実行してください（Python 3が必要です）。

```
cd fgo-scene-maker
python3 build_assets.py
```

これで `assets-data.js` / `font-data.js` が新しい内容で上書きされます。

## ライセンス・クレジット

- 同梱している画像・動画素材（`assets/`以下）はすべてオリジナル制作物です。
- フォントは GenEi Fonts（源暎ラテミン）を使用しています。
- コード（HTML/CSS/JS/Pythonスクリプト）は現時点で明示的なライセンスは設定していません。改変・再配布等を行いたい場合は作者までご連絡ください。

## 今後の拡張の余地

- **動画としての保存**：全体をCanvasに直接描画する方式のため、`canvas.captureStream()` + `MediaRecorder`で録画機能を足すのは技術的に可能
- **セリフの文字送り（タイプライター）演出**：未実装（現在NEXTアイコンは常時表示/非表示の切替のみ）

## うまく動かないとき

ブラウザによっては、ローカルファイルの扱いに関するセキュリティ設定の違いでPNG書き出しが失敗する可能性もゼロではありません。
その場合は、フォルダごとどこかに置いた状態で、ターミナルから以下のようにローカルサーバーを立てて
`http://localhost:8000/` にアクセスしてみてください。

```
cd fgo-scene-maker
python3 -m http.server 8000
```

(Node.jsをお使いの場合は `npx serve` でも代用できます）
